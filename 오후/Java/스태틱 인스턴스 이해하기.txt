인스턴스 - 동적공간
동적공간 => 실행 중에 할당되는 것
	=> 객체 생성시마다 생겨나는 공간
	=> 인스턴스 공간에는 스태틱 사용가능
	=> 스태틱은 파일 로드된 시점에 이미 존재하니까

static - 정적 공간
정적공간 => 프로그램 로드된 시점(시작시점)에 할당
	=> 시작 시에 생긴 이미 정해진 공간
	=> 스태틱 공간에서는 인스턴스 사용 불가능
	=> 스태틱 생긴 시점에 인스턴스 없으니까

컴퓨터 구조
<ram>
	<프로그램>
		코드 -> 함수, 메서드
		스택 -> 변수, 상수, 정적배열
		정적데이터 -> 변수 상수 배열
	<사용자>
		힙
