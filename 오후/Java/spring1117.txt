*런타임과 컴파일 타임

개발자는 소스코드르르 작성 후 컴파일이라는 과정을 통해 기계어코드(010101)로 변환되어 실행 가능한 프로그램이 되며, 이러한 과정을 컴파일타임이라고 함
컴파일 과정을 마친 후 동작되어지는 때를 런타임이라고 함

1. 컴파일 : 컴파일러가 컴퓨터가 .java를 읽을 수 있도록 .class 번역해 놓는 과정
	* 컴파일 오류 = 신택스 오류(오타발견), 타입체크 오류
2. 런타임환경 : 컴퓨터가 컴파일 과정이 끝나고 파일을 읽는것(실행)
	@는 런타임환경에서 실행
	* 런타임 오류 = 사칙연산 오류, null참조 오류, 메모리 부족 오류

컴파일과 인터프리터
컴파일(Java) : 책한권을 전체 다 읽고 해석
인터프리터(JavaScript) : 책 한줄씩 읽고 해석
